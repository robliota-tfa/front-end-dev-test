JavaScript/NodeJS

For this test, you'll write a node module in CommonJS (no ES6 modules, sorry) with a method that validates whether or not a given string is a palindrome. Assume that the string will contain ascii text and numbers. Your code should ignore non-alphanumeric characters, as in evaluating full sentence palindromes which may contain spaces and punctuation.

There is a file in src/javascript-test/palindrome.js

  • Expose a method there called "isPalindrome" that should:
    • Return true when a String is a palindrome
    • Return false when a String is not a palindrome
    • Return false when a String is empty
    • When passed a parameter that is not a String, throws an Error, or an object that is an instance of one, with a descriptive message about what went wrong
